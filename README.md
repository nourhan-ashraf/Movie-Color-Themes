# 🎬🎨 Movie Color Themes
## VS Code Extension

Bring the cinematic vibes of your favorite movies directly into VS Code! 

This extension includes multiple **movie-inspired color themes** for VS Code, allowing you to code in style.  

## Included Themes

- **City of Stars** – Inspired by *La La Land*, dreamy purple, blue, and yellow tones.  
- **In the Mood for Love** – Romantic red and gold palette from *In the Mood for Love*.  
- **The Grand Budapest** – Light or dark pinks with teal accents, whimsical and stylish.  

## Features

- Multiple themes in **one extension**  
- Light and dark modes (depending on theme)  
- High-contrast options for better readability  

## Installation

1. Open VS Code  
2. Go to the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)  
3. Search for **Movie-Palettes-Extension**  
4. Click **Install**  

Or manually install from the VSIX file if you have it.  

## Usage

1. Press `Ctrl+Shift+P` / `Cmd+Shift+P` to open the Command Palette  
2. Type `Color Theme` → select `Preferences: Color Theme`  
3. Pick your favorite movie-inspired theme
